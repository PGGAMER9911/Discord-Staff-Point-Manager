# Discord-Staff-Point-Manager
A production-grade Discord bot for managing staff points with atomic transactions, audit-grade history, and secure Supabase integration.


Discord Staff Point Manager is a production-ready Discord.js v14 bot designed to manage staff points securely and reliably.

It features atomic point updates using PostgreSQL (Supabase), role-based command permissions, full audit-grade history logs, and DM-only history exports for privacy.

Built with scalability, security, and real-world abuse prevention in mind.

• Slash command based staff point management  
• Atomic point updates using PostgreSQL RPC functions  
• Full audit-grade points history (passbook-style)  
• Secure Supabase backend with RLS enabled  
• ID-based permission system (no role spoofing)  
• Cooldown & abuse prevention  
• DM-only history exports for privacy  
• Production-ready Discord.js v14 architecture
